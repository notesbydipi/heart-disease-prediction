# Heart Disease Prediction Using Deep Learning

This project focuses on the prediction of heart disease using Deep Learning hybrid models. Different architectures such as CNN+LSTM, CNN+BiLSTM, and CNN+Attention were implemented and compared to improve prediction accuracy and performance.

## Technologies Used
- Python
- Deep Learning
- TensorFlow
- Keras
- Pandas
- NumPy
- Matplotlib

## Features
- Data preprocessing and normalization
- Deep learning model training
- Hybrid model implementation
- Accuracy and performance comparison
- Early heart disease prediction

## Models Used
- CNN + LSTM
- CNN + BiLSTM
- CNN + Attention

## Outcome
The hybrid deep learning models achieved high accuracy and improved performance for heart disease prediction.
